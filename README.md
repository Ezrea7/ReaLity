# Xray

基于当前 `xray.sh` 的轻量化 Xray 管理脚本仓库骨架。

## 目标

- 基于现有菜单式 `xray.sh` 继续演进
- 保持轻量、稳定、高效
- 不照抄其他仓库的大型命令体系
- 逐步仓库化、模块化、可发布

## 当前状态

当前仓库为第一版骨架，核心脚本来自现有 `xray.sh`。
后续将逐步拆分为：

- `install.sh`
- `xray.sh`
- `src/*.sh`
- `.github/workflows/release.yml`
